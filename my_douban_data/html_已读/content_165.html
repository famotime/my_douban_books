  • Python编程 : 从入门到实践

    [美] 埃里克·马瑟斯 / 袁国忠 / 人民邮电出版社 / 2016-7-1 / 89.00元
    2019-07-29 读过

    我曾在大学里学过C语言选修课，不过那已经是差不多二十年前了。研发曾是我大学毕业时心目中的理想岗位，但结果阴差阳错去了其他岗位，之后就再也没碰过编程。 前段时间心血来潮，报了一个0基础学Python的在线课程。嗯，果然零基础也能上手？然后觉得边讲故事边学编程的方式，有趣固然是有趣，却嫌有点低幼，进度也不自由。那就自己看书吧。 我选了这本在豆瓣上评价最高的《编程从入门到实践》作为第一本入门书。花了一周业余时间把前半部分基础语法看完了，感觉书上讲解的都差不多看懂了，就是记不住，自己解题也有点懵。当然编程不能光看书，要多用多实践才能融会贯通，这道理我懂。这几天尝试写了几个简单的小程序，Python果然强大，感觉打开了新世界的大门。 人生在世，总是有很多遗憾，补得一个是一个吧。

    纸质版 72.70元

  • 蒸汽火车头

    [比利时] 冯索瓦·史奇顿（François Shuiten） / 任可心 / 北京联合出版公司 / 2017-4 / 99.80元
    2019-07-20 读过

    写实素描的画风，阴影部分可以看到整齐细密的排线，是我喜欢的风格。 蒸汽火车和火车司机只是时代滚滚向前时抛下的那么多事物中的一个缩影，当代社会互联网和AI将抛下更多的传统行业。历史总是重演，因为人性亘古不变；但技术永远向前，改变了就再也回不来了。 书中结尾处给了一个貌似驶向光明前景的未来，但细想想就算老司机真找到了他陪伴一生的“美人儿”，又能开到哪里去，又能做什么呢？最好的结局不过是在博物馆里谋一席之地。但一件实用的事物在刚被淘汰时只会被弃之如履，只能在人们淡忘很久之后，才可能因为久远而加上文化、历史、情感的标签，从另一个视角变成艺术的东西。当事人是怎么都等不到了。

    纸质版 65.50元

  • 你今天真好看

    [美]莉兹•克里莫 / 周高逸 / 天津人民出版社 / 2015-8 / 39.00元
    2019-07-08 读过

    画风很简单，甚至简陋，是动画片《辛普森一家》的风格（作者是该片动画剧组工作人员）。故事有童趣，清新暖萌。但豆瓣评分高达8.7，还是有点出乎我的意料。也许大家都在成人世界里受过太多伤害了，所以希望在童真世界里暂避风雨？对我来说，读完本书后有个蓬蓬勃勃的愿望从心里长起来：在绘画和文字都压缩到最简的情况下，也许我也可以画本图画书？只要平日里多留心，积攒点生活小乐趣，感觉表达不是大问题。

    纸质版 29.30元

  • 小狗钱钱 : 引导孩子正确认识财富、创造财富的“金钱童话"

    [德] 博多·舍费尔 / 王钟欣、余茜 / 南海出版公司 / 2009-2-1 / 20.00元
    2019-07-06 读过

    书中的“我”给小狗取名“钱钱”的时候，妈妈不同意，但爸爸却觉得这个名字很有趣，他说：“这样的话，我们呼唤‘钱’，然后‘钱’就来了，很不错嘛！”从最后的结果看，这名字真是个好彩头，但所有的一切都要从行动开始。而行动力的前提是要有行动的方法和思路，方法和思路来自于主动的思考，思考的动力来自于明确的财务目标和自信……这么一直追溯上去，最后的根源是你希望借助财务成功达成的人生目标，以及你有多看重这个目标。大多数人如果没有迫到眼前的危机，对人生意义之类的大哉问只怕是退避三舍，下班之余只求片刻闲暇和种种小确幸，所以财务自由就无从谈起。从这个角度来说，清晰的梦想目标是理财成功之道，明智的投资原则是理财成功之术。我们呼唤‘钱’，然后‘钱’会不会来，取决于我们发自内心的呼声有多大。

    纸质版 15.40元

  • 642件可写的事 : 停不下来的创意冒险

    美国旧金山写作社、San Francisco Writers'Grotto / 徐阳 / 四川文艺出版社 / 2019-2 / 55.00元
    2019-06-23 读过

    也许你像我一样，早就有练习写作的想法，但又总是在下笔前不知道从哪里开始，那么这本书可以把退缩的理由灭掉一个。这是一本写作的命题集，可以随便翻到这本书的某一页随便挑一个题目就开始写。快速翻完了这本书，因为它不是用来读的，而是用来写作实践的，除了题目外就是等待你填写的空白。如果要把书上的题目都写完才算完成使命，那想想642个题目，这本书应该能读一辈子吧？ 记录了几个题目，也想随便写写看。就像另一本教导创意写作的书《写出我心》中所提到的，“Now. please. go. write your asses off.现在就开始吧 开始写 写到骨头里。”，不等待灵感，不管写的好坏，只管去写，写的过程就是目标。行动起来，才有后续。

  • 雨

    [马来西亚] 黄锦树 / 四川人民出版社 / 2018-3 / 38.00元
    2019-06-19 读过

    雨、胶林、少年、女孩、劳作的人、艰辛的生活……这是我读后留下的一些印象。几篇关于同一家人境遇的系列作品风格最为突出，相同的名字、相同的家庭关系、相同的环境，表明这就是同一家人，但是前后故事里的境遇并不一致，有的是父亲先遭难，有的是儿子早夭，梦境和现实交织，时间线交错，鬼神与人世共存，一个魔幻多雨的世界。 我觉得这本书适合拍电视剧，现代悬疑但又是偏魔幻类的，很多故事都有画面感。但每个故事的结尾并不都能用逻辑释疑，有的故事干脆在中途就戛然而止。比如《归来》中二舅讲的故事很荒谬，似乎他在巫医的诱导下可以进入不同维度的空间，时间流逝极为缓慢，在那里过了一辈子，苏醒过来只是打了个瞌睡而已；好似做梦，但梦境又有不同的人在异世界的境遇佐证，又像真的活了这一遭。这些想法十分迷人。

  • 为什么读经典

    [意大利] 伊塔洛·卡尔维诺 / 黄灿然、李桂蜜 / 译林出版社 / 2006-8 / 25.00元
    2019-06-13 读过

    这是一本书评集，如书名所述，评论的书籍都是经典之作。但我绝大部分都没有听说过，若不是我太孤陋寡闻，就是这些所谓的经典都已经离现代普罗大众太远了。没读过原著，就来读作者颇为高深的学术性评论，真是让人晕头转向。作者说他的书评受人好评的原因之一是他常引用原文，但就我看来仍是引的太少，而评的太多。 也有少数几本被评论吸引而想读原著的书：《奥德赛》、弗朗索瓦·蓬热的《事物的声音》、博尔赫斯的《小径分岔的花园》。

    纸质版 18.80元

  • 国富国穷

    [美] 戴维·S·兰德斯 / 门洪华 / 新华出版社 / 2010-1 / 62.00元
    2019-06-09 读过

    在中美贸易战的时代大背景下，这是一本应景的书，两周时间断断续续看完了，感觉视野开阔不少。 作者一直在强调社会文化、人的作用，节俭、诚实、耐心、顽强，如果国民没有这些优良品质，自由并不一定导向繁荣昌盛，就像非洲一众从殖民地脱身出来的独立国家一样；意外的财富终究会流失，就像曾经辉煌一时的西班牙、葡萄牙、荷兰一样。

    纸质版 43.80元
