  • 拉戈·云奇1：继承者&W集团 : 继承者&W集团

    [比利时]让·范阿姆 编、菲利普·弗朗克 绘 / 李婧 / 北京联合出版公司 / 2016-9 / 49.80元
    2019-12-08 读过

    线描+水彩涂色风格的绘本，画的很精美漂亮。故事也挺吸引人，一个被公司高管杀害的亿万富翁，将身后百亿美元资产留给了一个无人知晓的无名小子。为什么选他做继承人？为什么此前一直保密不为人知？一个毛头小子如何上位，降服一群老资格的商界大佬？杀人凶手又是谁？ 故事情节符合逻辑，问题最终都得到了解答，但没有平铺直叙到底，始终留有悬念。故事推进还穿插了童年回忆的时间线，感觉适合改写为剧本，拍成电影。 “为什么之前一直把他藏起来呢？为什么要保密？” ——“隐姓埋名的做法至少让他度过了一个正常的童年，不像一般的豪门子弟，一个个都长坏了。”

  • Python数据分析基础

    2019-11-28 读过

    数据分析是Python的几大应用场景之一，挺好的入门书。内容涵盖了Python语言基础、CSV/Excel文件的读写和数据处理、SQLite数据库的基本操作、Matplotlib可视化图表简介，以及一些分析实例。其中对CSV/Excel文件操作部分颇为经典，先讲一遍使用基本模块的操作方法，再讲一遍使用Pandas的操作方法，讲解的很细致，对初学者颇为友好。

  • Python数据可视化之matplotlib实践

    刘大成 / 电子工业出版社 / 2018-9 / 59.00元
    2019-11-23 读过

    作为入门参考书，还是不错的。分门别类的给出很多示例，方便速查，比起看官方英文文档轻松很多。前后行文稍微有点重复，比如函数介绍和常见图表。 在当当看的电子书，最多只能跨两屏摘录笔记，想把笔记摘录导出，结果代码挤成一团不说，而且导出的不是完整笔记，每条摘录超出约200字（中文或字母）的内容就变成了“...” 当当的电子书资源丰富，但读书笔记的体验做得不如多看阅读。多看可以默认划线，而不是像当当那样必须先选段落再划线，而且可以跨多页内容划线，导出摘录笔记的内容也是完整的。这几点深得我心。

    纸质版 59.00元

  • 银河铁道之夜

  • 世界历史很有趣：袁腾飞讲日本史

    袁腾飞 / 民主与建设出版社 / 2014-10 / 袁腾飞
    2019-11-03 读过

    自从《明朝那些事儿》大火之后，不少人仿效其作者“当年明月”的风格通俗讲史，袁腾飞是其中比较出名的。既然是通俗讲史，历史故事和段子当然很重要。袁腾飞在自序中把自己写作的风格称之为“史话体”，“结合学院派和戏说体的长处，在叙述一国历史大事的同时，照顾细节，讲故事、讲段子。”当然这是他在自圆其说，为了照顾普罗大众的喜好，行文总体风格还是偏向戏说一些。比如，对于日本历史上众多名人骚客的风流韵事、花边新闻，不论是传说还是史实，大概一个都没有放过，着实废了一番笔墨。谁让老百姓爱看呢！

    纸质版 20.50元

  • 生活三部曲：生活的样子

    [法] 让-克劳德·弗洛克 / 周克希 / 北京联合出版公司 / 2016-7 / 60.00元

    大开本，绘画风格简单，总共几十页一下就翻完了。内容是父亲和女儿生活的一些片段，可以说温馨，但说不上多么让人触动，生活的样子本来就是很简单平凡的嘛。 我当前还是喜欢纷繁复杂的细致画风，简单风的图画书总觉得平淡了，但有一点好，看着觉得自己作为一个普通人也能画。虽然直到现在还从没动过手。也许先可以积累一些生活轶事和感悟文字吧，以后再配图。

  • 东大爸爸写给我的日本史

    [日] 小岛毅 / 王筱玲 / 北京联合出版公司 / 2018-8 / 68.00元
    2019-10-16 读过

    相比一些专门讲故事的历史书，这本书并没有那么引人入胜。即使有故事也是概略性的，如果读者不熟悉相关日本历史或者跟中国历史无涉，则多半对其中描述的事件无感。 值得一提的是作者对待史学的态度，并没有把颂扬本国历史，提升民族自豪感之类的政治诉求带入到历史研究。而是结合日本历史记载以及来自邻国中国的历史资料，来揣度更符合逻辑的历史真相。有些历史迷案则摆出两边的证据让读者自己评判。 对生活在当时的人们来说，所处的时代是纷繁复杂的；后人看前史，固然可以有更通盘的视角，更明晰的事件走向，但历史的真相也会随时间淡化消逝。道德规范和朝代的变迁，会让同一件事的观感迥异。为了王朝的正统性和鼓励国民的爱国主义，一些历史不免被刻意拔高或者刻意掩盖。 从来就没有所谓真实的历史，历史的记述更多的由胜利者书写。

    纸质版 53.00元

  • 以图代言 : 用漫画讲述产品故事

    Kevin Cheng / 江业华、七印部落 / 华中科技大学出版社 / 2015-3-25 / 46.00
    2019-10-15 读过

    “用漫画讲述产品故事”--嗯，这是一个好想法。可是，我不会画画怎么办呀？这也许是多数人心里冒出的第一个疑问吧。所以作者花了很大的篇幅来打消人们的担心，并用一些实例来展示如何用简笔画来设计和展示产品。 读后有一些启发，但要有好的效果，还是需要有一些设计经验的。万事总是开头难吧，有兴趣就不妨下手试试。漫画的好处在于打消人们的防备心理，只要能吸引到人的注意力，大概就算成功了一半。换个角度说，这本书就是作者推出的产品，如果作者的漫画能说服读者亲自下手试一试，才算成功。所以，这是一个可以论证自身的书，成不成功就由读者你来说了算吧。

    纸质版 43.30元

  • 笨办法学 Python 3

    [美] Zed A. Shaw / 王巍巍 / 人民邮电出版社 / 2018-6-1 / CNY 59.00
    2019-10-15 读过

    这是很多人推荐的一本Python新手入门书，所谓“笨办法”指的是不取巧、不走捷径，踏踏实实练基本功。所以作者非常强调读者不要直接拷贝书中的代码，而是手工把代码敲到自己的电脑里去，培养编程思维和手感。 相比这本书学习Python编程的课程设计，我更认同作者在书中表达的一些编程理念，比如： 关于学习新编程语言的最后一件要记住的事情就是：别当一个“蠢游客”。“蠢游客”就是那种去了一个国家旅游，然后回来抱怨那儿的饭不好吃的人。“为什么这个白痴国家连汉堡都买不到？”当你学习一种新编程语言时，不要假设它的工作方式太蠢，它只是不同而已，只有接受它你才能学会它。 嗯，心有戚戚焉。

    纸质版 49.40元

  • 京都漫步 : 穷游·最世界「京都」特辑

    骆仪（主编） / 北京联合出版公司 / 2017-1 / 58元

  • 图解物联网

    ［ 日］ NTT DATA集团、河村雅人、大塚纮史、小林佑辅、小山武士、宫崎智也、石黑佑树、小岛康平 / 丁灵 / 人民邮电出版社 / 2017-4 / 59.00元
    2019-09-26 读过

    日本人写的一本物联网开发科普书，7个主要作者里面有5个是机器人开发领域的，这算是一种日本科研特色吧，所以机器人开发当然要独立成一章。作为一本科普书，内容涉及面颇广，当然也就不能指望提供什么深入见解了，定位就是科普嘛。 图解是本书特点，也是吸引人的地方。如果内容涉及到大量元素且相互间有复杂关系或逻辑，那么图画比文字能更好的让人理解。但就本书而言，我觉得图解必要性不是很大。比如把列举了三个要点的文字，转换成加了关键词的三个圆圈，再贴上几个具象化图标（如wifi图标和移动网络图标，如路由器代表设备终端等），对这样的图解，也许直接阅读文字效率还高些。

    纸质版 49.40元

  • 希腊罗马神话 : 永恒的诸神·英雄·爱情与冒险故事

  • 自学是门手艺 : 没有自学能力的人没有未来

    李笑来 / 电子工业出版社 / 2019-6 / 99元
    2019-08-29 读过

    很久以前读过李笑来的《把时间当做朋友》，对书本内容已经记不起来了，印象里是一部跟成功学相关的鸡汤文集。为什么留下来这个印象，大概因为这是一本讲道理的书，当年读来并不生动，讲道理又讲得不生动的书对我而言都是鸡汤文。 最近因为对Python有兴趣，读了李笑来与Python编程相关的新作《自学是门手艺》，感觉挺有帮助，顿时对他的观感改变不少。不同于一般的鸡汤文作家，只会讲道理编故事，李笑来看来是个学以致用的实干家，或许之前对他的印象都是错的。我想抽空把他的其他书找来都读一遍，包括多年前读过却没有留下痕迹的《把时间当做朋友》。

    纸质版 47.88元
