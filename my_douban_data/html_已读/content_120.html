  • 画的秘密

    [法] 马克-安托万·马修 / 贾梦 / 北京联合出版公司 / 2016-1 / 60.00元
    2020-11-10 读过

    世界上有很多不同风格的漫画，有的故事曲折，有的画风精美，而本书的作者则擅长带给你一个奇妙的想法。作者围绕一个核心概念写故事，没有前因后果、没有复杂情节，纯纯粹粹的让你惊奇和思考。不是思考人生的那种思考，而是奇思妙想的那种思考。 主人公获得了过世老友的一副神奇的赠画——严格的说，这幅画不是老友送给他的，而是他在一整个堆满艺术品中自己挑选出的纪念品，但老友显然笃定知道他会作出这样的选择。画中细节可以无限放大，进入另一个世界的想法跟作者的另一部作品《3秒》有点类似，但没有像《3秒》走得那么远那么夸张。 这是一个幻想故事，当然不可能在现实生活中发生。好讲好写，但不容易让人相信，口花花谁不会呢，说得越天花乱坠越难相信。而作者居然能在画面上把这样一个设计呈现出来，应该花了不少心思，让人赞叹。了不起。

    纸质版 47.30元

  • 再一次

    [加] 布莱恩·李·奥马利 / 梁梓聪 / 北京联合出版公司 / 2017-7 / 68.00元
    2020-10-21 读过

    一个吃后悔药的故事，还掺杂了一个“房灵”的传说，有点混搭。 然后颇有教义的告诉你，这个世界上没有后悔药，或者说你吃了后悔药最终会更后悔，就像那句很有哲理的话说的：“世界上所有的礼物都被暗中标好了价格”。 我记得有一部叫《时间遥控器》好莱坞电影，主人公一直快进跳过不愉悦的时光，有点类似的讲故事风格。我倒是觉得写一本吃了后悔药，然后修正所有缺憾，成为人生赢家的故事，更显得不落窠臼呢。

    纸质版 68.00元

  • 边城 : 2013最新修订纪念典藏版

    沈从文 / 武汉出版社 / 2013-9 / 35
    2020-09-24 读过

    沈从文的这部名著，多次拿起又放下，这次总算是读完了。 文字是很好的，素淡而雅致，没动用华丽的辞藻，却能显出平常文字的优美来。似乎民国期间的作家，多有这种本事。民风淳朴的风土人情，无论是老船夫不受人恩惠的脾气，还是船总顺顺周到的待人接物，甚至那些做皮肉生意妇人的痴梦，这些浓郁的人情味，是如今都见识不到的了。 故事算不上多么曲折，山城里姑娘小伙的爱情故事，一些意外和波折，是所有世道里的人都可能经历的生活苦难。在文末，沈老爷子还是留了一点希望，比我想象的要仁慈。 相比起来，书里另一篇《都市一妇人》中的爱情故事更让我震动，尤其是揭晓谜底的一刻。残酷真相才是生活的本色啊。

  • 跟菲利大叔学手绘：创意速写 : 创意速写

    [德] 菲利克斯·沙因伯格 / 郭璐 / 上海人民美术出版社 / 2017-1-1 / 59.00元
    2020-09-24 读过

    在照相机发明之后，再追求画的像就有点徒劳了。抽象画是一条路子，速写也许是另一条。作者在这本书里传授的主要不是绘画技巧，而是如何看待速写，如何通过速写表达情绪，如何展示你自己眼中的真实，如何获得绘画的乐趣。 相比当代艺术家经常在网上搜索一副图片，然后照着图片作画，作者更希望绘画者能走出家门，用自己的双眼去观察真实的世界，然后用绘画来表达自己的观察所得及体会。那是跟照相机拍下的影像，完全不同的一种东西，更自我，更聚焦，甚至更真实。

    纸质版 29.60元

  • 快速阅读术

    【日】印南敦史 / 王宇新 / 中信出版社 / 2017-4-1 / 39.00元
    2020-08-17 读过

    作者提倡“流水式”阅读，让书籍内容从心中“流过”，只要“流过”便有意义。借此来消除读者希望把所有读过的内容都理解记住的执念。 另外，作者提出“一行采集”、“一行精华”、“一行评论”的读书方法，也是为了尽量降低读书难度，就像微博相比博客吸引了更多的人群参与一样。第一步首先培养阅读兴趣，有了兴趣后面逐步会有更多的高质量的输出，将是自然而然的事，但第一步首先要降低对自己的要求和期望。 无论做什么事，放松心态，才能走的更远。

    纸质版 30.40元

  • 海狼

    [法] 里夫·雷布斯 编绘 / 崔书凝 / 北京联合出版公司 / 2017-3 / 78.00元
    2020-08-06 读过

    读后印象深刻的有哪些呢? 首先是海狼船长，那个凶狠强壮、无视道德，却又熟读莎士比亚的男人，不时还爆出一两句深含哲理的话；还有那个粗鄙心狭的厨子，既让人痛恨，又让人觉得可怜；还有那两名勇敢反抗、多次逃亡的船员，最后消失在海浪中…… 还有"我"这一趟奇特曲折的冒险旅程： 轮船失事，被幽灵船救起，被迫成为海员； 在船上从杂役干到大副，经历航海中各种变故，远离文明的野蛮生活； 与一名因海难上船的女作家逃离幽灵船，在荒岛上度过了一段甜蜜时光； 阴阳差错在荒岛上又遇上搁浅的幽灵船和独自一人奄奄一息的海狼； 在即将返回文明世界的希望中，海狼死去，爱人消失…… 当然还有全书精细生动的画风，无论是刻画人物动作还是表达人物性格都很有神韵，跟杰克伦敦的小说原著相得益彰。

    纸质版 60.80元

    2020-07-10 读过

    《无星之国》 男权国家，男人打仗，女人干杂务，漂亮女人做当权者玩物；女权国家，还是男人打仗，男人干杂务，优秀男人也得做当权者玩物；比起来还是男人比较苦啊！最后两国领导人居然被外人劝说几句就结束战争，放下仇恨，迎来和平共处——看来这不是科幻故事，只能算童话故事。 《阿尔弗洛洛星球欢迎你》 银河帝国法律竟然没有要求对外星人赶尽杀绝，还要保障原住民的权利，原住民有权要回土地？ 看来还是童话啊。 《主子的疯鸟》 寓意最深刻的一篇，故事也比较完整。 对于一个永不餍足、能够操纵心灵的外星怪兽，该怎么对付它呢？像是寓言故事里那些想尽办法躲避猫的老鼠，谁也不愿意去当那只在猫脖子上挂铃铛的老鼠——现实生活里逆行的英雄总是少数。

  • 无证之罪 : 推理之王1

    紫金陈 / 湖南人民出版社 / 2018-6 / 49.00元
    2020-07-06 读过

    小说写到一小半的时候，罪案的谜底已经猜到了，悬念只是故事会怎么结局。在中国背景下大概只能是法网恢恢疏而不漏，不可能像好莱坞电影一样，罪犯竟能逃脱法律的制裁，逃之夭夭。 解方程的比喻挺精当的：普通罪案用公式解，复杂罪案用代入法尝试再校验。解决生活中的难题大概也得用代入法，边试边总结，不保证有法子一定成功。 但是最初的疑点是：嫌犯虽然有钱，但生活简朴不应该买豪车？牵强了些。

    纸质版 23.70元

  • 白夜行

    [日] 东野圭吾 / 刘姿君 / 南海出版公司 / 2013-1-1 / 39.50元
    2020-07-01 读过

    一直保持到最后阶段的核心悬念、精巧的叙事结构、复杂的人物关系网络，这些也许能让一部小说成为一时的畅销书，但要成为公认的经典之作，那还不够。大家对本书的评价如此之高，我想很重要的原因是这个故事的象征意义。“ 我的天空没有太阳，总是黑夜，但并不暗，因为有东西代替了太阳。虽然没有太阳那么明亮，但对我来说已经足够。凭借这份光，我便能把黑夜当成白天。我从来就没有太阳，所以不怕失去。 ”似乎是这种超越了爱情和时间的长久羁绊，引起了人们持久不断的热情讨论。说到底，这世上每个人都是孤独的，都渴望着有一个无论发生了什么都不离不弃，始终陪伴左右，照耀自己的太阳。

    纸质版 30.80元
