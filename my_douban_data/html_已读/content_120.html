  • 惊爆点

    (日) 松本大洋等编绘 / 沈丽凝 / 湖南美术出版社 / 2018-8 / 99.80元
    2020-04-29 读过

    13个漫画短篇故事，全球14个漫画家（其中有个人负责封面设计而非故事创作），我还是比较喜欢日本漫画家的作品。比如： 松本大洋（ Taiyo Matsumoto）——《花子的屁》，每个人的一瞬间都有不同的意义； 浦泽直树( Naoki Urasawa)——《孤胆英雄》，开头跟女人的沟通很写实，最后的反转让人忍俊不禁。 小池桂一 ( Keiichi Koike)——《鱼》，精细的画风，让人惊艳的奇思妙想。

  • 孩子们的诗

    果麦 编 / 浙江文艺出版社 / 2017-10 / 68.00元
    2020-04-20 读过

    孩子们写的诗，人们当然是看重他们的童真：童言稚语、未经雕琢、浑然天成。但这些挑拣出来的诗作，当然是从大人眼里看到的童真，是筛选过滤过的童真。 也许当孩子拿起笔的时候，已经不是自然的状态，孩子也会去迎合大人的期望，更别提有些诗作也许会经过大人的修饰完善。所以我以为，孩子的诗最好不是自己写的，而是旁人在其日常生活中记录的言语欢笑，这样方得自然。 我分不清楚哪些是自然的童真，哪些是故作稚语，所以我还是以成人的眼光来鉴赏，那些记录了意料之外却合情合理的奇思妙想，那些记录了平平常常却让人会心一笑的生活况味，才是我的心头好。比如下面这首： 《 等待 》 沙发上 一个女人悠哉的吸着烟 另一个女人 悠闲地玩着手机 还有一个 十一岁的小女孩 在翻滚 谁也看不出 她们在焦急的等待

  • 漫长的告别 : 插图珍藏版

    [美] 雷蒙德·钱德勒 / 姚向辉 / 海南出版社 / 2018-9 / 69.00
    2020-04-04 读过

    最惨的就是那个特里前恋人——艾琳·韦德——“像一个美梦一样”的绝代佳人。按照钱德勒的情节设计，艾琳因为怨恨，枪杀了跟她前老公结婚，又跟她现老公鬼混的女人；特里为了保护这个前任老婆，砸坏了现任老婆的面目来掩盖枪伤，并甘愿背黑锅逃离，坐实了自己的杀人罪名。然后，特里的黑帮朋友废了牛劲为他做局假死。然后美女艾琳明明已经脱罪了非要折腾，招惹马洛、勾引马洛，还杀了她现任老公企图嫁祸马洛，布了好大一个局。然而马洛没上钩，反而把一切都揭穿，逼得美女自杀。 所以，特里为了保护前恋人背了这个杀人黑锅，算是白背了？马洛为了给众人眼里已经死去的特里留个清白名声，破坏了他知己朋友作局的初衷。所以，特里是该怪自己考虑不周详呢，还是怪马洛狗拿耗子多管闲事，或者怪他的前恋人聪明反被聪明误？

  • 在细雨中呼喊

    余华 / 南海出版公司 / 1999-1 / 18.00元
    2020-03-19 读过

    余华的作品，笔下的生活真实而残酷。所谓“仓禀实而知礼节”，当温饱都无法保证的时候，人生很多故事都是以残酷而丑陋的形式而展开的。比如，“我”的父亲嫌祖父光吃饭不干活，盼着他早点死；比如，冯玉青受生活所迫而做皮肉生意，被抓进了劳改农场，留下一个小孩子孤苦无依的独自生活…… 这些特定时代背景下的小人物故事，因年代久远而显得魔幻，但情感的力量仍然让人可以深信不疑这些是曾经发生过的真实故事。因为情感的真实，很多平平常常的语句，称不上优美华丽，但读起来打动人心。 还有些是永恒的情感，比如少年的思春，受同龄人的排挤的烦恼；很多情感在成年后淡忘了，但又被文字勾起。

    纸质版 18.00元

  • Python游戏编程快速上手 第4版

    Al Sweigart / 李强 / 人民邮电出版社 / 2017-10-1 / 69.00元
    2020-03-09 读过

    这本书的作者跟《Python编程快速上手 : 让繁琐工作自动化》是同一个人，行文比较通俗易懂，中间还会对一些基础语法做讲解，充分照顾了初学者。前面几章的文本游戏比较简单，从第15章Reversegam游戏开始就有点难度了。对于初学者来说，编写游戏程序相比其他利用编程完成的练习，通常是一个更大的挑战，一个游戏更像是一个完整的应用系统，方方面面要考虑周全，对系统设计和逻辑思维要求较高。但这个过程充满乐趣，不仅能学习编程知识，成品还能供人娱乐，也算圆了一个童年梦想，何乐而不为呢。

    纸质版 46.30元

  • 像计算机科学家一样思考Python（第2版）

    [美] Allen B. Downey / 赵普明 / 人民邮电出版社 / 2016-7 / 49.00元
    2020-03-07 读过

    这本书从最基础的概念讲起，但不止于介绍Python语言本身，而是侧重于培养编程思维，只不过书中的例子都用Python语言来实现而已。相对更侧重于编程的道，而不是Python的术，全书没有涉及太高深的技巧，覆盖的Python知识点也不如其他书那么全面，但学会如何使用编程来解决问题的思路比学会一门编程语言具备更广泛的适用性，因此在一众Python入门书中独树一帜，值得细细揣摩。 书里有不少英文“字谜”相关的有趣习题，在我看来，这是人类的渊博知识在计算机面前不堪一击的典型例子，在人工智能已经威胁并在事实上取代众多传统工作岗位的当下，值得去思考身为人类的我们应该如何面对机器智慧，以及在自我完善的道路上如何做出取舍。

    纸质版 32.90元

  • Python学习手册（第4版）

    [美] Mark Lutz / 李军、刘红伟 / 机械工业出版社 / 2011-4 / 119.00元
    2020-03-03 读过

    当我读到这本书的884页，看到一句话：“祝贺你！如果你学习到这里，就可以把自己自己当成真正的Python程序员了”，让我分外激动。我终于也算是程序员了呀！ 我从开始阅读到全书读完一遍（其中1-24章看了两遍，25-35章介绍类和面向对象编程部分看了一遍），刚好花了半年的时间。中间也穿插看了一些其他的Python入门书。就其厚度来说，这本书对初学者不甚友好，足以让多数小白从入门读到放弃。但反过来厚也是它的优势，正因为厚才讲的细致，才能让人深入理解Python这门语言的特性。一般来讲，这么厚的书只能当手册查阅用，这类书无非是把官网资料的内容重复一遍，介绍命令参数、再举几个例子，但这本书的行文却不是这样，而是真的把一些知识点掰开了揉碎了讲清楚，所以如果耐得住性子，非常值得从头到尾读一遍。

    纸质版 82.00元

  • Python快速入门

    [美] 娜奥米·塞德（ Naomi Ceder ） / 戴旭 / 人民邮电出版社 / 2019-8 / 89
    2020-02-16 读过

    这是一本2019年8月份新出版的Python学习入门书籍，内容比较全面，从数组、字典等基本数据类型，到类/面向对象编程、常见标准库的使用，到数据库、数据分析等应用实例，都覆盖到了。每个知识点基本都是先概要介绍，然后讲解一两个简短的例子来加深理解，行文比较清晰简洁。 如果你觉得一开始没有信心直接尝试厚达1000页的《Python学习手册》，可以把这本书作为替代。我是花了几个月时间断断续续先读完了《Python学习手册》，然后读这本书几天就快速翻完了，作为Python知识点手册来复习回顾，感觉也挺不错。

    纸质版 77.40元

  • 超级快速阅读

    [德] 克里斯蒂安·格吕宁 / 郝湉 / 中信出版社 / 2011-4 / 35.00元
    2020-02-08 读过

    正如书名所示，这本书教得是快速阅读方法，侧重于知识的获取和记忆。从有效学习的角度来看，因为每个人的时间和精力有限，学习的第一步首先应该是筛选学习对象，然后才是针对重点内容的学习和记忆。这本书针对的是第二步。 具体来看，快速阅读有几个要点： 1 借助手指等辅助工具，训练自己扩展视野，减少默读习惯，加快阅读速度（同时保持阅读理解程度不变甚至有提高）； 2 借助联想、想象以及地点桩等技巧，运用多种感官，提升知识点记忆效率； 3 按照艾宾浩斯遗忘曲线制定复习计划，将学习到的知识转入长期记忆。

    纸质版 26.30元

  • 从Python开始学编程

    Vamei / 电子工业出版社 / 2016-11-24 / CNY 49.00
    2020-02-06 读过

    前面几章是面向小白的，甚至不止是编程小白，而是计算机小白，所以一开始门槛很低。但是到了讲解对象的章节开始，涉及的知识就有点高级了，但也写得清晰易懂。最好的部分是最后一章讲解函数式编程，这是其他Python入门教程中较少涉及的内容，让我对这个主题有了初步的认识。

    纸质版 32.90元

    [比利时] 让·范阿姆 编、菲利普·弗朗克 绘 / 李婧 / 北京联合出版公司 / 2017-9 / 49.80元
    2020-02-05 读过

  • 蓝莓上尉：遗失的金矿 : 遗失的金矿

    [比] 让-米歇尔·沙利耶 编、[法] 墨比斯 / 曹艳艳、赵晓宇 / 文化发展出版社 / 2018-1 / 60.00元
    2020-02-04 读过

    感觉好像看了一部美国西部片，偏写实的画风展现了一个未完全在法制管辖之下的边陲小镇，跟日本漫画注重细节描绘的都市风迥异。刻画的一众人物形象性格鲜明，有本领高强的牛仔，有凶狠毒辣的杀手，有老奸巨猾的骗子，还有经不住金钱诱惑但仍然重情重义的老酒鬼。这其中，奸诈狡猾的“德国佬”，多次被困却总能耍手腕逃离生天，给我留下的印象最深。 虽然是漫画，文字量还不少，讲述了一个完整的西部探宝冒险故事。剧作家专门撰写故事，漫画家专事绘画，两者分工合作、相得益彰，为这本书增色不少。

    纸质版 36.00元
