  • 昆仑6 : 天道卷

    凤歌 / 团结出版社 / 2006-1 / 22.00元
    2015-05-23 读过

    终于看完了这套前传加正传7本书的大部头，梁萧不但文武双全，而且算学、机械、阵法冠绝天下，可算是武侠小说中少数几个走出中国江湖，游历世界的奇才。作者对中国传统文化颇有研究，行文有金庸的风格。后面几本略压抑，不如前面几本看得爽快。

    纸质版 22.00元

  • 昆仑3 : 破城卷

    凤歌 / 团结出版社 / 2005-10 / 20.00元
    2015-05-23 读过

    我们看武侠小说的时候通常会把自己代入到主角身上，借此体验主角或刺激或香艳的江湖生涯，心底暗爽。但在《昆仑3》中要以身代入主角梁萧却有些困难。因为在本书中梁萧一反“侠之大者，为国为民”的传统侠义观念，非但没有率领中原群雄力敌蒙古大军，反而帮助蒙古大军连续攻城拔寨，连南宋最重要的屏障襄阳城也因梁萧的妙计而被攻陷，南宋百姓为此被荼毒无数。这对传统观念实在是一个挑战，我在理性上赞赏作者跳出窠臼的情节设置，但在感性上却对主角梁萧明珠暗投心存芥蒂，这种违和感伴随阅读始终，降低了一些YY的乐趣。

    纸质版 46.80元

  • 昆仑2 : 纯阳卷

    凤歌 / 团结出版社 / 2005-9 / 20.00元
    2015-05-23 读过

    秘籍异宝、练功升级、情海生波，都是武侠小说的固定套路，也是吸引读者的地方。这一部的情节中规中矩，作者文笔不错，读来还是挺爽快的。另外，书中对古代算学、阵势、历法的描写，算是相比传统武侠小说有新意的地方。

    纸质版 36.40元
