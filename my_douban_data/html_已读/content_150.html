  • 以图代言 : 用漫画讲述产品故事

    Kevin Cheng / 江业华、七印部落 / 华中科技大学出版社 / 2015-3-25 / 46.00
    2019-10-15 读过

    “用漫画讲述产品故事”--嗯，这是一个好想法。可是，我不会画画怎么办呀？这也许是多数人心里冒出的第一个疑问吧。所以作者花了很大的篇幅来打消人们的担心，并用一些实例来展示如何用简笔画来设计和展示产品。 读后有一些启发，但要有好的效果，还是需要有一些设计经验的。万事总是开头难吧，有兴趣就不妨下手试试。漫画的好处在于打消人们的防备心理，只要能吸引到人的注意力，大概就算成功了一半。换个角度说，这本书就是作者推出的产品，如果作者的漫画能说服读者亲自下手试一试，才算成功。所以，这是一个可以论证自身的书，成不成功就由读者你来说了算吧。

    纸质版 43.30元

  • 笨办法学 Python 3

    [美] Zed A. Shaw / 王巍巍 / 人民邮电出版社 / 2018-6-1 / CNY 59.00
    2019-10-15 读过

    这是很多人推荐的一本Python新手入门书，所谓“笨办法”指的是不取巧、不走捷径，踏踏实实练基本功。所以作者非常强调读者不要直接拷贝书中的代码，而是手工把代码敲到自己的电脑里去，培养编程思维和手感。 相比这本书学习Python编程的课程设计，我更认同作者在书中表达的一些编程理念，比如： 关于学习新编程语言的最后一件要记住的事情就是：别当一个“蠢游客”。“蠢游客”就是那种去了一个国家旅游，然后回来抱怨那儿的饭不好吃的人。“为什么这个白痴国家连汉堡都买不到？”当你学习一种新编程语言时，不要假设它的工作方式太蠢，它只是不同而已，只有接受它你才能学会它。 嗯，心有戚戚焉。

    纸质版 49.40元

  • 京都漫步 : 穷游·最世界「京都」特辑

    骆仪（主编） / 北京联合出版公司 / 2017-1 / 58元
    2019-10-07 读过

    书并不厚，内容还是不错的，不同于一般的打卡式行程介绍，更偏向于文化散文和访谈录。重点看了第一部分关于寺庙的文章，毕竟是古都嘛，每个古寺都有很多历史故事；后面部分“本地人的声音”也是一个不错的视角。 本来是想当京都旅行攻略看的，结果在出发前没找到时间，出游结束回国后才翻了翻。如果能有时间提前翻一遍的话，也许我会改变一些行程，去几个书中介绍的冷门景点寻访故事吧，但兴之所至盲游的感觉也挺好。根据京都府的官方统计，京都足足有2493座神社寺庙、908个都市公园呢，短短一个国庆假期是逛不完的，有机会再去。

  • 图解物联网

    ［ 日］ NTT DATA集团、河村雅人、大塚纮史、小林佑辅、小山武士、宫崎智也、石黑佑树、小岛康平 / 丁灵 / 人民邮电出版社 / 2017-4 / 59.00元
    2019-09-26 读过

    日本人写的一本物联网开发科普书，7个主要作者里面有5个是机器人开发领域的，这算是一种日本科研特色吧，所以机器人开发当然要独立成一章。作为一本科普书，内容涉及面颇广，当然也就不能指望提供什么深入见解了，定位就是科普嘛。 图解是本书特点，也是吸引人的地方。如果内容涉及到大量元素且相互间有复杂关系或逻辑，那么图画比文字能更好的让人理解。但就本书而言，我觉得图解必要性不是很大。比如把列举了三个要点的文字，转换成加了关键词的三个圆圈，再贴上几个具象化图标（如wifi图标和移动网络图标，如路由器代表设备终端等），对这样的图解，也许直接阅读文字效率还高些。

    纸质版 44.00元

  • 希腊罗马神话 : 永恒的诸神·英雄·爱情与冒险故事

    (美) 伊迪丝·汉密尔顿 / 余淑慧 / 中信出版社 / 2017-2 / 49
    2019-09-16 读过

    看完了这本书，算是把此前零零碎碎听到读到的希腊神话梳理了一遍。希腊的神固然神通广大，经常干出很出格的事情（比如宙斯到处猎艳），但仍不失人性（即使是宙斯猎艳之举也可说是人性使然，大概有太多作者把难以遂愿的男女之事按在宙斯头上了）。在希腊神话故事中，就算是神也无法摆脱命运的驱使，像人一样经历不如意之事，像人一样有喜怒哀乐。

  • 自学是门手艺 : 没有自学能力的人没有未来

    李笑来 / 电子工业出版社 / 2019-6 / 99元
    2019-08-29 读过

    很久以前读过李笑来的《把时间当做朋友》，对书本内容已经记不起来了，印象里是一部跟成功学相关的鸡汤文集。为什么留下来这个印象，大概因为这是一本讲道理的书，当年读来并不生动，讲道理又讲得不生动的书对我而言都是鸡汤文。 最近因为对Python有兴趣，读了李笑来与Python编程相关的新作《自学是门手艺》，感觉挺有帮助，顿时对他的观感改变不少。不同于一般的鸡汤文作家，只会讲道理编故事，李笑来看来是个学以致用的实干家，或许之前对他的印象都是错的。我想抽空把他的其他书找来都读一遍，包括多年前读过却没有留下痕迹的《把时间当做朋友》。

    纸质版 82.90元

  • Head First HTML与CSS（第2版）

    Elisabeth Robson、Eric Freeman / 徐阳、丁小峰 / 中国电力出版社 / 2013-9 / 98.00元
    2019-08-22 读过

    前段时间学了Python想写个爬虫实践一下，解析网页需要能大致看明白网页源码中HTML标签含义，于是找了这本《Head First HTML与CSS（第2版）》来看，有名的深入浅出系列。 厚达700多页的一本HTML及CSS入门书，两天周末时间就基本翻完了。之前接触过HTML，还是在十多年前上大学的时候，那会儿设计网页是一个时髦的技术活，不少人借此挣学费，比家教赚得多。时隔这么久，我还能一口气看完基本没有卡顿，一方面说明学习HTML不难（看懂HTML源码简单，当然要把网页设计的漂亮就难了），另一方面这本书也恰如其分的完成了带领读者入门的任务。凭这点要给个好评。

    纸质版 75.50元

  • 名画之谜：历史故事篇

    [日]中野京子（Nakano Kyoko） / 俞隽 / 中信出版社 / 2015-11 / 42.00元
    2019-08-21 读过

    自古以来，历史都是胜利者书写的历史。作者开篇讲到的理查三世似乎就是一例，战败后遗体被剥光衣服示众，还被敌对者用笔写成了为皇位不择手段心思恶毒的人，反正死人已经不能从地底下跳出来辩白。在中国历史上，著名的商纣王也以暴虐闻名，也有历史学家认为是其后的周朝统治者故意抹黑，但历史太久远，真相早已湮没不复得见。 除了史书，文学作品更有着颠倒黑白、化凡入圣的能力。比如三国演义中勇猛的赵云、忠义的关云长，神机妙算的诸葛亮，都已成为后世传说中熠熠生辉的人物，而白脸的曹操、小心眼的周瑜，则成为衬托主角的反派典型。理查三世正是这么不幸，在大文豪莎士比亚的同名剧作中被塑造为个性十足的反派角色，这下可好，在老百姓眼里永世不得翻身了。 这本书的版面太小，对应文字描述找画面都费眼睛，看着不爽。

    纸质版 27.70元
