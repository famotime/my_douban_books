  • 像计算机科学家一样思考Python（第2版）

    [美] Allen B. Downey / 赵普明 / 人民邮电出版社 / 2016-7 / 49.00元
    2020-03-07 读过

    这本书从最基础的概念讲起，但不止于介绍Python语言本身，而是侧重于培养编程思维，只不过书中的例子都用Python语言来实现而已。相对更侧重于编程的道，而不是Python的术，全书没有涉及太高深的技巧，覆盖的Python知识点也不如其他书那么全面，但学会如何使用编程来解决问题的思路比学会一门编程语言具备更广泛的适用性，因此在一众Python入门书中独树一帜，值得细细揣摩。 书里有不少英文“字谜”相关的有趣习题，在我看来，这是人类的渊博知识在计算机面前不堪一击的典型例子，在人工智能已经威胁并在事实上取代众多传统工作岗位的当下，值得去思考身为人类的我们应该如何面对机器智慧，以及在自我完善的道路上如何做出取舍。

    纸质版 49.00元

  • Python学习手册（第4版）

    [美] Mark Lutz / 李军、刘红伟 / 机械工业出版社 / 2011-4 / 119.00元
    2020-03-03 读过

    当我读到这本书的884页，看到一句话：“祝贺你！如果你学习到这里，就可以把自己自己当成真正的Python程序员了”，让我分外激动。我终于也算是程序员了呀！ 我从开始阅读到全书读完一遍（其中1-24章看了两遍，25-35章介绍类和面向对象编程部分看了一遍），刚好花了半年的时间。中间也穿插看了一些其他的Python入门书。就其厚度来说，这本书对初学者不甚友好，足以让多数小白从入门读到放弃。但反过来厚也是它的优势，正因为厚才讲的细致，才能让人深入理解Python这门语言的特性。一般来讲，这么厚的书只能当手册查阅用，这类书无非是把官网资料的内容重复一遍，介绍命令参数、再举几个例子，但这本书的行文却不是这样，而是真的把一些知识点掰开了揉碎了讲清楚，所以如果耐得住性子，非常值得从头到尾读一遍。

    纸质版 82.00元

  • Python快速入门

    [美] 娜奥米·塞德（ Naomi Ceder ） / 戴旭 / 人民邮电出版社 / 2019-8 / 89
    2020-02-16 读过

    这是一本2019年8月份新出版的Python学习入门书籍，内容比较全面，从数组、字典等基本数据类型，到类/面向对象编程、常见标准库的使用，到数据库、数据分析等应用实例，都覆盖到了。每个知识点基本都是先概要介绍，然后讲解一两个简短的例子来加深理解，行文比较清晰简洁。 如果你觉得一开始没有信心直接尝试厚达1000页的《Python学习手册》，可以把这本书作为替代。我是花了几个月时间断断续续先读完了《Python学习手册》，然后读这本书几天就快速翻完了，作为Python知识点手册来复习回顾，感觉也挺不错。

    纸质版 89.00元

  • 超级快速阅读

    [德] 克里斯蒂安·格吕宁 / 郝湉 / 中信出版社 / 2011-4 / 35.00元
    2020-02-08 读过

    正如书名所示，这本书教得是快速阅读方法，侧重于知识的获取和记忆。从有效学习的角度来看，因为每个人的时间和精力有限，学习的第一步首先应该是筛选学习对象，然后才是针对重点内容的学习和记忆。这本书针对的是第二步。 具体来看，快速阅读有几个要点： 1 借助手指等辅助工具，训练自己扩展视野，减少默读习惯，加快阅读速度（同时保持阅读理解程度不变甚至有提高）； 2 借助联想、想象以及地点桩等技巧，运用多种感官，提升知识点记忆效率； 3 按照艾宾浩斯遗忘曲线制定复习计划，将学习到的知识转入长期记忆。

    纸质版 26.30元

  • 从Python开始学编程

    Vamei / 电子工业出版社 / 2016-11-24 / CNY 49.00
    2020-02-06 读过

    前面几章是面向小白的，甚至不止是编程小白，而是计算机小白，所以一开始门槛很低。但是到了讲解对象的章节开始，涉及的知识就有点高级了，但也写得清晰易懂。最好的部分是最后一章讲解函数式编程，这是其他Python入门教程中较少涉及的内容，让我对这个主题有了初步的认识。

    纸质版 49.00元

  • 蓝莓上尉：遗失的金矿 : 遗失的金矿

    [比] 让-米歇尔·沙利耶 编、[法] 墨比斯 / 曹艳艳、赵晓宇 / 文化发展出版社 / 2018-1 / 60.00元
    2020-02-04 读过

    感觉好像看了一部美国西部片，偏写实的画风展现了一个未完全在法制管辖之下的边陲小镇，跟日本漫画注重细节描绘的都市风迥异。刻画的一众人物形象性格鲜明，有本领高强的牛仔，有凶狠毒辣的杀手，有老奸巨猾的骗子，还有经不住金钱诱惑但仍然重情重义的老酒鬼。这其中，奸诈狡猾的“德国佬”，多次被困却总能耍手腕逃离生天，给我留下的印象最深。 虽然是漫画，文字量还不少，讲述了一个完整的西部探宝冒险故事。剧作家专门撰写故事，漫画家专事绘画，两者分工合作、相得益彰，为这本书增色不少。

    纸质版 48.20元

  • 雪中悍刀行 1 : 西北有雏凤

    烽火戏诸侯 / 江苏文艺出版社 / 2013-9-1 / 25.00元
    2020-01-31 读过

    很久没看武侠小说了。前段时间看技术书看得有点累，于是拿了这本《雪中悍刀行》换换口味，打算在春节期间放松一下。这本书名气挺大，网上有不少人评论说好看，不比一般的网络爽文。 结果没看下去，只坚持到第1本书的大概2/3篇幅。感觉作者写得故作高深，既没有看到傻小子奇遇记的爽利，也没有看到曲折解谜的勾人，尝试了两次，对作者勾画的江湖，实在投入不进去，最终放弃。 上学的时候，武侠小说曾经是我的最爱。小时候汽车站门口有一个借书摊，一本书租一天5毛钱，被人借得多的书都被磨得卷边了。暑假里父母才允许我借小说看，犹记得借到一本心仪已久的武侠书，迫不及待雀跃回家看书的心情。现在是很难再体验到了。

    纸质版 24.70元

  • Python GUI设计：tkinter菜鸟编程

    清华大学出版社
    2020-01-15 读过

    tkinter是Python语言默认的GUI编程标准库，用来写一些小工具的图形界面，够用。看了网上一些评价，似乎对tkinter最大的抱怨是文档可读性差，对接口和用法描述不清楚。网上的一些贴文也都是零零碎碎的，不成体系。 因此，买了这本介绍tkinter编程的书，2019年8月出版，全彩印刷，内容比较详尽。从头到尾翻了一遍，大概看了一下例子，了解了有哪些控件，等需要用时再翻书查就好。好像大家对GUI编程兴趣不高啊，豆瓣上对这本书的信息都收录不全，至今还没有人发表过评价。就让我来写第一个评论吧，作为工具书，还不错的。

    纸质版 67.20元

  • 3秒

    [法] 马克-安托万·马修 / 后浪漫 / 湖南美术出版社 / 2019-5 / 72.00元
    2020-01-12 读过

    我相信这本漫画来自一个突发的灵感：跟随一束光在极短时间内去看事件全景。但是从灵感落地到实际的作品，需要大量精心的设计，类似于拍电影的长镜头，拍起来很辛苦，很少能做到从头到尾一镜到底。开头部分在同一个场景，跟随光和镜面，从不同角度查看人物、现场，真是很酷的设计。感觉如同首次看黑客帝国电影里的360度全景拍摄，很惊艳。 光束在不同镜面间反射成像，第一个镜面是人的眼球，倒影着他眼前的手机，手机摄像头的玻璃镜面映照着身后一个拿枪指着他的男人……这样一层层映照下去，镜头不断探入、放大图像，几至无穷。想想你随后几十页看到的景象，都来自最初眼球中的景象倒影，如果光和镜面记录的影响分辨率是无穷的，这就在理论上是可实现的。这真是一个迷人的想法。

    纸质版 72.00元

  • Python网络数据采集

    米切尔 (Ryan Mitchell) / 陶俊杰、陈小莉 / 人民邮电出版社 / 2016-3-1 / CNY 59.00
    2020-01-08 读过

    我学习Python编程的第1本书是《Python编程：从入门到实践》，第2本书就是这本《Python网络数据采集》。爬虫应该是Python几个应用方向里上手最快，最方便新手展示成果的。前3章比较实用，学了就可以拿些小网站练手，去爬点东西。 因为成书较早，本书对动态网站爬取利器Selenium只是略略一提，没有展开，因为当时的Selenium版本还不支持Python3。而且因为是老外写的书，随后几章讲Google和Twitter API之类的，虽然方法相通，但跟中国国情有些不符。后面讲解了爬取数据之后的文件存储、数据库，OCR、自然语言识别等内容，虽然也有用，但对于新手来说有些过于高深了。所以跳着看了一些就搁置了，转去阅读国内爬虫高手写的书。直到近6个月后才返过头来，终于完整看完一遍。

    纸质版 51.30元

  • 了不起的Markdown

    毕小朋 / 电子工业出版社 / 2019-8 / 65.00元
    2019-12-21 读过

    Markdown语言规则简单，可以很快上手。我半年前开始用Markdown写文章，在网上搜了些教程看看，感觉基本够用。看这本书是为了更全面系统的了解一下Markdown规则，算是进阶。 书中所讲的Markdown语法知识，我大多已经知道了。数学公式、流程图这些高级（低频）应用，书中没有详细介绍，我暂时也用不上。有一个插入emoji表情的功能，之前不了解，挺有意思。书中后面一部分是介绍支持Markdown写作的编辑器，比如使用VS code和Jupiter写文章，之前没有试过，后续可以试试。 总体看，如果是首次接触Markdown，读这本书比较有价值，讲得比较全面，可以快速上手，也可以作为手册备查。如果是已经有过Markdown写作经验，获得新知识比较有限，快速翻翻就好。

    纸质版 54.44元

  • 树莓派Python编程入门与实战

    勃鲁姆 (Richard Blum)、布莱斯纳罕 (Christine Bresnahan) / 陈晓明、马立新 / 人民邮电出版社 / 2016-8-1 / 69
    2019-12-16 读过

    树莓派4买回来就积灰了，为了物尽其用就找了几本书看。这本书虽然书名以树莓派打头，但书中主要讲的是Python编程，至于编程环境是基于树莓派还是普通PC机其实都无所谓。 前面几章关于Python语法和数据结构部分比较基础，包含了一些小细节小技巧；关于面向对象编程部分讲解比较清晰易懂，对小白来说值得一看；后面关于GUI编程、游戏编程、Web编程等没展开，基本上就是蜻蜓点水了。 这本书在豆瓣上评价很少，我觉得作为入门书质量还行，可以快速浏览一遍，温故知新。

    纸质版 57.80元
