  • 写作这回事 : 创作生涯回忆录

    [美]斯蒂芬·金 / 张坤 / 上海译文出版社 / 2009.08 / 28.00元
    2015-05-23 读过

    这是一本讲写作的书，但并不是一堂写作课。作者在书里谈写作，也谈生活。读一读大师的成长历程，也许更有助益。如果你想发家致富，写作显然不是一个好的选择，大师在成名前的生活也是清贫而阴霾的，而且丝毫看不到成功的希望。但如果你发自心底的热爱写作，但又始终犹豫着要不要开始，那么作者已经给所有的这样的文学爱好者颁发了许可证。而且“故事犹如埋在地下的化石，它们是遗迹，属于一个未被发现但已经存在的世界。作家的工作就是利用工具把每个故事尽量完好无损的从地里挖出来”，我想作者的这种说法多少能减少一些写作新手内心的忧惧和自我怀疑。那么现在就开始吧，记得要乐观的面对失败。

    纸质版 9.80元

  • 道济群生录

    张万康 / 广西师范大学出版社 / 2014-6 / 38.00
    2015-05-23 读过

    听说这本书得过2011年的台湾文学金典奖，慕名下载试读。小说风格确实很独特，像是在说书，又像是在主持台湾综艺节目，还让我想起来小时候有个大哥哥经常给我们几个小孩子随口瞎掰讲故事，各种典故乱入。作者是爽了，下笔信马由缰，口沫肆意纵横，可是好像还没挠到我的痒处，说不上有优美的文笔，也说不上扣人心弦的情节，要说想象力倒更似恶搞。总之可以看个热闹，但不是太对胃口。很想知道这本书在台湾畅不畅销，除了那些文学大师之外，台湾老百姓喜不喜欢这种风格。

    纸质版 22.80元

  • 昆仑3 : 破城卷

    凤歌 / 团结出版社 / 2005-10 / 20.00元
    2015-05-23 读过

    我们看武侠小说的时候通常会把自己代入到主角身上，借此体验主角或刺激或香艳的江湖生涯，心底暗爽。但在《昆仑3》中要以身代入主角梁萧却有些困难。因为在本书中梁萧一反“侠之大者，为国为民”的传统侠义观念，非但没有率领中原群雄力敌蒙古大军，反而帮助蒙古大军连续攻城拔寨，连南宋最重要的屏障襄阳城也因梁萧的妙计而被攻陷，南宋百姓为此被荼毒无数。这对传统观念实在是一个挑战，我在理性上赞赏作者跳出窠臼的情节设置，但在感性上却对主角梁萧明珠暗投心存芥蒂，这种违和感伴随阅读始终，降低了一些YY的乐趣。

    纸质版 46.00元
