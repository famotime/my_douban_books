  • 算法神探 : 一部谷歌首席工程师写的CS小说

    [美] 杰瑞米·库比卡 / 啊哈磊、李嘉浩 / 电子工业出版社 / 2017-2 / 65
    2020-06-14 读过

    这本书的目标读者当然是初学算法的编程小白，希望借助一些讲解生动的书来入门。但这本书明显犯了喧宾夺主的毛病，算法知识点太少，而且被层层包裹在一个并不生动有趣，反而有些故弄玄虚的儿童故事里。正经学东西的话，知识密度太低；你如果想看故事，那明显是找错了地方。 相比起来，另一本算法入门书——《算法图解》，阅读感受要好得多。

    纸质版 56.60元

  • 小鸟睡在我身旁

    2020-06-03 读过

    这本书写了两对母女的生活小事：“我”和女儿，以及母亲和“我”。前者主要是写女儿成长趣事，后者主要是写母亲对家庭的爱护和奉献，间或也提及了一些“我”的生活。 如今当一个专职写字的不知名作家，如果没有家人的支持和其他生活来源，生活大概是清苦的，好在生活的幸福感更多依赖情趣和美感，而不是金钱。这些对生活的细致观察、甘于普通的生活态度，如果能潜移默化的被孩子吸收，也许比一笔庞大的遗产，更能让孩子抵御住往后生活的风雨。

    纸质版 31.40元

  • 氯的滋味

    [法] 巴斯蒂安·维韦斯 / 张鑫佳 / 北京联合出版公司 / 2016-4 / 88.00元
    2020-05-17 读过

    这是一个恋爱故事吗？算不上，刚刚互有好感，暧昧尚未滋生，故事就结束了。情节动人吗？故事平平常常，称之为情节不如说是一段情绪。 作者是法国人，画于2007年，不算太久远。这样的漫画能在西方世界获得好评，我也是有点意外，法国人不是很开放很主动的嘛。几次相遇，互有好感，连名字和电话都没要，这种纯纯的爱恋发生在东方世界才正常。 画风不太精细，好像是一个未经绘画训练的普通人画的日常故事，这样反而让人一开始就不去苛求画工，专注于体会其中的氛围和情绪。 有几个镜头是从第一视角展现的，让人有种投入的感觉，比如：仰泳的时候，整个视野都是玻璃屋顶，然后边上有人从视野边界。啊，是那个我刚才还在满场搜寻却没找到的女孩子，她竟然主动跟我打招呼……

    纸质版 68.60元

  • 伤心咖啡馆之歌

    [美]卡森·麦卡勒斯 / 文泽尔 / 华东师范大学出版社 / 2018-3-30 / 38.00元
    2020-05-14 读过

    这是一个令人悲伤的三角恋故事，我爱你，你爱他，他爱我，首尾相接却永不得满足。 因爱生恨的事情并不少见，天堂和地狱只有一线之隔。艾米丽娅的不知人情世故似乎是后来所有悲剧的开端，她接受了马文·梅西的求婚却实际拒绝了与他共同生活，还狠狠羞辱了他。后来马文·梅西和驼背人离开小镇前干尽了坏事，他们拿走了所有值钱的东西，砸了酿酒的机器，烧了作坊，还做出了一盘有毒的玉米粥，摆在了咖啡馆的柜台上。这些事情有多出格，大概显示了马文·梅西恨的有多深，而此前爱的又有多深。

    纸质版 22.10元

  • 万物：创世

    [德] 延斯·哈德 / 王遥路、陈亮 / 北京联合出版公司 / 2015-4 / 188.00元
    2020-05-08 读过

    这个绘本在豆瓣上评分高达8.9分，得过国际漫画节大奖，不过不是我的菜。 内容很有野心，第一册从宇宙大爆炸开始，一直画到恐龙灭绝，人类登场。画风不是很精细，内容斑杂，暗藏了很多古人探索解释世界的理论和原画，但以西方宗教古典为主，我不是考据狂，并没有进一步探究的想法。如果要说知识，似是而非的绘画，当然比不上详细的文字说明。我前不久读的《万物简史》传递的信息量就要大得多，依我看也更加有趣。

    纸质版 109.60元

  • 孩子们的诗

    果麦 编 / 浙江文艺出版社 / 2017-10 / 68.00元
    2020-04-20 读过

    孩子们写的诗，人们当然是看重他们的童真：童言稚语、未经雕琢、浑然天成。但这些挑拣出来的诗作，当然是从大人眼里看到的童真，是筛选过滤过的童真。 也许当孩子拿起笔的时候，已经不是自然的状态，孩子也会去迎合大人的期望，更别提有些诗作也许会经过大人的修饰完善。所以我以为，孩子的诗最好不是自己写的，而是旁人在其日常生活中记录的言语欢笑，这样方得自然。 我分不清楚哪些是自然的童真，哪些是故作稚语，所以我还是以成人的眼光来鉴赏，那些记录了意料之外却合情合理的奇思妙想，那些记录了平平常常却让人会心一笑的生活况味，才是我的心头好。比如下面这首： 《 等待 》 沙发上 一个女人悠哉的吸着烟 另一个女人 悠闲地玩着手机 还有一个 十一岁的小女孩 在翻滚 谁也看不出 她们在焦急的等待

  • 漫长的告别 : 插图珍藏版

    2020-04-04 读过

    最惨的就是那个特里前恋人——艾琳·韦德——“像一个美梦一样”的绝代佳人。按照钱德勒的情节设计，艾琳因为怨恨，枪杀了跟她前老公结婚，又跟她现老公鬼混的女人；特里为了保护这个前任老婆，砸坏了现任老婆的面目来掩盖枪伤，并甘愿背黑锅逃离，坐实了自己的杀人罪名。然后，特里的黑帮朋友废了牛劲为他做局假死。然后美女艾琳明明已经脱罪了非要折腾，招惹马洛、勾引马洛，还杀了她现任老公企图嫁祸马洛，布了好大一个局。然而马洛没上钩，反而把一切都揭穿，逼得美女自杀。 所以，特里为了保护前恋人背了这个杀人黑锅，算是白背了？马洛为了给众人眼里已经死去的特里留个清白名声，破坏了他知己朋友作局的初衷。所以，特里是该怪自己考虑不周详呢，还是怪马洛狗拿耗子多管闲事，或者怪他的前恋人聪明反被聪明误？

  • 在细雨中呼喊

    余华 / 南海出版公司 / 1999-1 / 18.00元
    2020-03-19 读过

    余华的作品，笔下的生活真实而残酷。所谓“仓禀实而知礼节”，当温饱都无法保证的时候，人生很多故事都是以残酷而丑陋的形式而展开的。比如，“我”的父亲嫌祖父光吃饭不干活，盼着他早点死；比如，冯玉青受生活所迫而做皮肉生意，被抓进了劳改农场，留下一个小孩子孤苦无依的独自生活…… 这些特定时代背景下的小人物故事，因年代久远而显得魔幻，但情感的力量仍然让人可以深信不疑这些是曾经发生过的真实故事。因为情感的真实，很多平平常常的语句，称不上优美华丽，但读起来打动人心。 还有些是永恒的情感，比如少年的思春，受同龄人的排挤的烦恼；很多情感在成年后淡忘了，但又被文字勾起。

    纸质版 18.00元

  • Python游戏编程快速上手 第4版

    Al Sweigart / 李强 / 人民邮电出版社 / 2017-10-1 / 69.00元
    2020-03-09 读过

    这本书的作者跟《Python编程快速上手 : 让繁琐工作自动化》是同一个人，行文比较通俗易懂，中间还会对一些基础语法做讲解，充分照顾了初学者。前面几章的文本游戏比较简单，从第15章Reversegam游戏开始就有点难度了。对于初学者来说，编写游戏程序相比其他利用编程完成的练习，通常是一个更大的挑战，一个游戏更像是一个完整的应用系统，方方面面要考虑周全，对系统设计和逻辑思维要求较高。但这个过程充满乐趣，不仅能学习编程知识，成品还能供人娱乐，也算圆了一个童年梦想，何乐而不为呢。

    纸质版 66.90元
