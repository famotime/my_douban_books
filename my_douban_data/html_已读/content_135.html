  • 雪中悍刀行 1 : 西北有雏凤

    烽火戏诸侯 / 江苏文艺出版社 / 2013-9-1 / 25.00元
    2020-01-31 读过

    很久没看武侠小说了。前段时间看技术书看得有点累，于是拿了这本《雪中悍刀行》换换口味，打算在春节期间放松一下。这本书名气挺大，网上有不少人评论说好看，不比一般的网络爽文。 结果没看下去，只坚持到第1本书的大概2/3篇幅。感觉作者写得故作高深，既没有看到傻小子奇遇记的爽利，也没有看到曲折解谜的勾人，尝试了两次，对作者勾画的江湖，实在投入不进去，最终放弃。 上学的时候，武侠小说曾经是我的最爱。小时候汽车站门口有一个借书摊，一本书租一天5毛钱，被人借得多的书都被磨得卷边了。暑假里父母才允许我借小说看，犹记得借到一本心仪已久的武侠书，迫不及待雀跃回家看书的心情。现在是很难再体验到了。

    纸质版 13.30元

  • Python GUI设计：tkinter菜鸟编程

    清华大学出版社
    2020-01-15 读过

    tkinter是Python语言默认的GUI编程标准库，用来写一些小工具的图形界面，够用。看了网上一些评价，似乎对tkinter最大的抱怨是文档可读性差，对接口和用法描述不清楚。网上的一些贴文也都是零零碎碎的，不成体系。 因此，买了这本介绍tkinter编程的书，2019年8月出版，全彩印刷，内容比较详尽。从头到尾翻了一遍，大概看了一下例子，了解了有哪些控件，等需要用时再翻书查就好。好像大家对GUI编程兴趣不高啊，豆瓣上对这本书的信息都收录不全，至今还没有人发表过评价。就让我来写第一个评论吧，作为工具书，还不错的。

    纸质版 57.70元

  • 3秒

    [法] 马克-安托万·马修 / 后浪漫 / 湖南美术出版社 / 2019-5 / 72.00元
    2020-01-12 读过

    我相信这本漫画来自一个突发的灵感：跟随一束光在极短时间内去看事件全景。但是从灵感落地到实际的作品，需要大量精心的设计，类似于拍电影的长镜头，拍起来很辛苦，很少能做到从头到尾一镜到底。开头部分在同一个场景，跟随光和镜面，从不同角度查看人物、现场，真是很酷的设计。感觉如同首次看黑客帝国电影里的360度全景拍摄，很惊艳。 光束在不同镜面间反射成像，第一个镜面是人的眼球，倒影着他眼前的手机，手机摄像头的玻璃镜面映照着身后一个拿枪指着他的男人……这样一层层映照下去，镜头不断探入、放大图像，几至无穷。想想你随后几十页看到的景象，都来自最初眼球中的景象倒影，如果光和镜面记录的影响分辨率是无穷的，这就在理论上是可实现的。这真是一个迷人的想法。

    2020-01-09 读过

    这应该是一本好书，虽然我没有给五星评价。书跟人一样，最美好的场景莫过于在合适的时间，合适的状态下，遇到合适的书。如果相遇的时间不对，感觉也就不一样了。对我来说，第一次接触Python的新鲜和兴奋感已经献给了《Python编程：从入门到实践》，初见只有一次，再也回不去了。

  • Python网络数据采集

    米切尔 (Ryan Mitchell) / 陶俊杰、陈小莉 / 人民邮电出版社 / 2016-3-1 / CNY 59.00
    2020-01-08 读过

    我学习Python编程的第1本书是《Python编程：从入门到实践》，第2本书就是这本《Python网络数据采集》。爬虫应该是Python几个应用方向里上手最快，最方便新手展示成果的。前3章比较实用，学了就可以拿些小网站练手，去爬点东西。 因为成书较早，本书对动态网站爬取利器Selenium只是略略一提，没有展开，因为当时的Selenium版本还不支持Python3。而且因为是老外写的书，随后几章讲Google和Twitter API之类的，虽然方法相通，但跟中国国情有些不符。后面讲解了爬取数据之后的文件存储、数据库，OCR、自然语言识别等内容，虽然也有用，但对于新手来说有些过于高深了。所以跳着看了一些就搁置了，转去阅读国内爬虫高手写的书。直到近6个月后才返过头来，终于完整看完一遍。

    纸质版 49.40元

  • 了不起的Markdown

    毕小朋 / 电子工业出版社 / 2019-8 / 65.00元
    2019-12-21 读过

    Markdown语言规则简单，可以很快上手。我半年前开始用Markdown写文章，在网上搜了些教程看看，感觉基本够用。看这本书是为了更全面系统的了解一下Markdown规则，算是进阶。 书中所讲的Markdown语法知识，我大多已经知道了。数学公式、流程图这些高级（低频）应用，书中没有详细介绍，我暂时也用不上。有一个插入emoji表情的功能，之前不了解，挺有意思。书中后面一部分是介绍支持Markdown写作的编辑器，比如使用VS code和Jupiter写文章，之前没有试过，后续可以试试。 总体看，如果是首次接触Markdown，读这本书比较有价值，讲得比较全面，可以快速上手，也可以作为手册备查。如果是已经有过Markdown写作经验，获得新知识比较有限，快速翻翻就好。

    纸质版 43.60元

  • 树莓派Python编程入门与实战

    勃鲁姆 (Richard Blum)、布莱斯纳罕 (Christine Bresnahan) / 陈晓明、马立新 / 人民邮电出版社 / 2016-8-1 / 69
    2019-12-16 读过

    树莓派4买回来就积灰了，为了物尽其用就找了几本书看。这本书虽然书名以树莓派打头，但书中主要讲的是Python编程，至于编程环境是基于树莓派还是普通PC机其实都无所谓。 前面几章关于Python语法和数据结构部分比较基础，包含了一些小细节小技巧；关于面向对象编程部分讲解比较清晰易懂，对小白来说值得一看；后面关于GUI编程、游戏编程、Web编程等没展开，基本上就是蜻蜓点水了。 这本书在豆瓣上评价很少，我觉得作为入门书质量还行，可以快速浏览一遍，温故知新。

  • 拉戈·云奇1：继承者&W集团 : 继承者&W集团

    [比利时]让·范阿姆 编、菲利普·弗朗克 绘 / 李婧 / 北京联合出版公司 / 2016-9 / 49.80元
    2019-12-08 读过

    线描+水彩涂色风格的绘本，画的很精美漂亮。故事也挺吸引人，一个被公司高管杀害的亿万富翁，将身后百亿美元资产留给了一个无人知晓的无名小子。为什么选他做继承人？为什么此前一直保密不为人知？一个毛头小子如何上位，降服一群老资格的商界大佬？杀人凶手又是谁？ 故事情节符合逻辑，问题最终都得到了解答，但没有平铺直叙到底，始终留有悬念。故事推进还穿插了童年回忆的时间线，感觉适合改写为剧本，拍成电影。 “为什么之前一直把他藏起来呢？为什么要保密？” ——“隐姓埋名的做法至少让他度过了一个正常的童年，不像一般的豪门子弟，一个个都长坏了。”

  • Python数据分析基础

    [美] Clinton W. Brownley / 陈光欣 / 人民邮电出版社 / 2017-8 / 69.00元
    2019-11-28 读过

    数据分析是Python的几大应用场景之一，挺好的入门书。内容涵盖了Python语言基础、CSV/Excel文件的读写和数据处理、SQLite数据库的基本操作、Matplotlib可视化图表简介，以及一些分析实例。其中对CSV/Excel文件操作部分颇为经典，先讲一遍使用基本模块的操作方法，再讲一遍使用Pandas的操作方法，讲解的很细致，对初学者颇为友好。

  • Python数据可视化之matplotlib实践

    刘大成 / 电子工业出版社 / 2018-9 / 59.00元
    2019-11-23 读过

    作为入门参考书，还是不错的。分门别类的给出很多示例，方便速查，比起看官方英文文档轻松很多。前后行文稍微有点重复，比如函数介绍和常见图表。 在当当看的电子书，最多只能跨两屏摘录笔记，想把笔记摘录导出，结果代码挤成一团不说，而且导出的不是完整笔记，每条摘录超出约200字（中文或字母）的内容就变成了“...” 当当的电子书资源丰富，但读书笔记的体验做得不如多看阅读。多看可以默认划线，而不是像当当那样必须先选段落再划线，而且可以跨多页内容划线，导出摘录笔记的内容也是完整的。这几点深得我心。

  • 世界历史很有趣：袁腾飞讲日本史

    袁腾飞 / 民主与建设出版社 / 2014-10 / 袁腾飞
    2019-11-03 读过

    自从《明朝那些事儿》大火之后，不少人仿效其作者“当年明月”的风格通俗讲史，袁腾飞是其中比较出名的。既然是通俗讲史，历史故事和段子当然很重要。袁腾飞在自序中把自己写作的风格称之为“史话体”，“结合学院派和戏说体的长处，在叙述一国历史大事的同时，照顾细节，讲故事、讲段子。”当然这是他在自圆其说，为了照顾普罗大众的喜好，行文总体风格还是偏向戏说一些。比如，对于日本历史上众多名人骚客的风流韵事、花边新闻，不论是传说还是史实，大概一个都没有放过，着实废了一番笔墨。谁让老百姓爱看呢！

    纸质版 20.50元

  • 生活三部曲：生活的样子

    大开本，绘画风格简单，总共几十页一下就翻完了。内容是父亲和女儿生活的一些片段，可以说温馨，但说不上多么让人触动，生活的样子本来就是很简单平凡的嘛。 我当前还是喜欢纷繁复杂的细致画风，简单风的图画书总觉得平淡了，但有一点好，看着觉得自己作为一个普通人也能画。虽然直到现在还从没动过手。也许先可以积累一些生活轶事和感悟文字吧，以后再配图。

  • 东大爸爸写给我的日本史

    [日] 小岛毅 / 王筱玲 / 北京联合出版公司 / 2018-8 / 68.00元
    2019-10-16 读过

    相比一些专门讲故事的历史书，这本书并没有那么引人入胜。即使有故事也是概略性的，如果读者不熟悉相关日本历史或者跟中国历史无涉，则多半对其中描述的事件无感。 值得一提的是作者对待史学的态度，并没有把颂扬本国历史，提升民族自豪感之类的政治诉求带入到历史研究。而是结合日本历史记载以及来自邻国中国的历史资料，来揣度更符合逻辑的历史真相。有些历史迷案则摆出两边的证据让读者自己评判。 对生活在当时的人们来说，所处的时代是纷繁复杂的；后人看前史，固然可以有更通盘的视角，更明晰的事件走向，但历史的真相也会随时间淡化消逝。道德规范和朝代的变迁，会让同一件事的观感迥异。为了王朝的正统性和鼓励国民的爱国主义，一些历史不免被刻意拔高或者刻意掩盖。 从来就没有所谓真实的历史，历史的记述更多的由胜利者书写。

    纸质版 68.00元
