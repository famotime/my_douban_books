  • 快速阅读术

    【日】印南敦史 / 王宇新 / 中信出版社 / 2017-4-1 / 39.00元
    2020-08-17 读过

    作者提倡“流水式”阅读，让书籍内容从心中“流过”，只要“流过”便有意义。借此来消除读者希望把所有读过的内容都理解记住的执念。 另外，作者提出“一行采集”、“一行精华”、“一行评论”的读书方法，也是为了尽量降低读书难度，就像微博相比博客吸引了更多的人群参与一样。第一步首先培养阅读兴趣，有了兴趣后面逐步会有更多的高质量的输出，将是自然而然的事，但第一步首先要降低对自己的要求和期望。 无论做什么事，放松心态，才能走的更远。

    纸质版 30.40元

  • 海狼

    [法] 里夫·雷布斯 编绘 / 崔书凝 / 北京联合出版公司 / 2017-3 / 78.00元
    2020-08-06 读过

    读后印象深刻的有哪些呢? 首先是海狼船长，那个凶狠强壮、无视道德，却又熟读莎士比亚的男人，不时还爆出一两句深含哲理的话；还有那个粗鄙心狭的厨子，既让人痛恨，又让人觉得可怜；还有那两名勇敢反抗、多次逃亡的船员，最后消失在海浪中…… 还有"我"这一趟奇特曲折的冒险旅程： 轮船失事，被幽灵船救起，被迫成为海员； 在船上从杂役干到大副，经历航海中各种变故，远离文明的野蛮生活； 与一名因海难上船的女作家逃离幽灵船，在荒岛上度过了一段甜蜜时光； 阴阳差错在荒岛上又遇上搁浅的幽灵船和独自一人奄奄一息的海狼； 在即将返回文明世界的希望中，海狼死去，爱人消失…… 当然还有全书精细生动的画风，无论是刻画人物动作还是表达人物性格都很有神韵，跟杰克伦敦的小说原著相得益彰。

    纸质版 33.50元

  • 亲历抗美援朝战争

    孟昭瑞 / 北京联合出版公司 / 2015-4 / 99.80元
    2020-07-30 读过

    这是一部摄影集，没有太多文字描写，很多故事要从照片里读出来。 封面选的是一张后勤部队手拉肩扛、翻山越岭，往前线送弹药给养的照片，在美军火力封锁下，后勤之路就是死亡之路，上甘岭战役期间据说能给前线送到一个苹果就给记功，可见后勤困难到了什么地步。这种情况下中国守军还能坚守不退，真是难以想象。 拍摄战争场面的照片少了一些，有点遗憾。

    纸质版 99.80元

  • 读库1804

    张立宪 主编 / 新星出版社 / 2018-8-26 / 30
    2020-07-13 读过

    《嘉靖律政风云》：作品要写得好看，当然最好是故事本身是个好故事，有冲突、有反转、还能品砸出一点世间的道理。 《伤痕》：时代的一粒灰，落到个人头上就是一座山。每个人都带着时代的印记，我们有幸不是生活在最艰难的时代。 《流浪广州》：世上有这么多人，每个人都有独一无二的际遇。有些非主流的生活方式，只要那个人自己不以为苦，便很好。 《私人文学史（续二）》：书籍极大丰富的今天，对于爱书人是一个幸福的时代，但对于培养爱书的习惯却是负累。就像食物短缺时最馋吃，一旦敞开供应，嘴巴就刁了，滋味没有了，注意力也分散了。 《比疯狂还疯狂的想象》：科学的尽头像是要跟宗教和神话接轨。

    纸质版 18.00元

  • 星际特工2

    [法]克里斯坦 编、梅齐埃 绘 / 孔秀云 / 北京联合出版公司 / 2017-6 / 99.80元
    2020-07-10 读过

    《无星之国》 男权国家，男人打仗，女人干杂务，漂亮女人做当权者玩物；女权国家，还是男人打仗，男人干杂务，优秀男人也得做当权者玩物；比起来还是男人比较苦啊！最后两国领导人居然被外人劝说几句就结束战争，放下仇恨，迎来和平共处——看来这不是科幻故事，只能算童话故事。 《阿尔弗洛洛星球欢迎你》 银河帝国法律竟然没有要求对外星人赶尽杀绝，还要保障原住民的权利，原住民有权要回土地？ 看来还是童话啊。 《主子的疯鸟》 寓意最深刻的一篇，故事也比较完整。 对于一个永不餍足、能够操纵心灵的外星怪兽，该怎么对付它呢？像是寓言故事里那些想尽办法躲避猫的老鼠，谁也不愿意去当那只在猫脖子上挂铃铛的老鼠——现实生活里逆行的英雄总是少数。

    纸质版 59.90元

  • 无证之罪 : 推理之王1

    紫金陈 / 湖南人民出版社 / 2018-6 / 49.00元
    2020-07-06 读过

    小说写到一小半的时候，罪案的谜底已经猜到了，悬念只是故事会怎么结局。在中国背景下大概只能是法网恢恢疏而不漏，不可能像好莱坞电影一样，罪犯竟能逃脱法律的制裁，逃之夭夭。 解方程的比喻挺精当的：普通罪案用公式解，复杂罪案用代入法尝试再校验。解决生活中的难题大概也得用代入法，边试边总结，不保证有法子一定成功。 但是最初的疑点是：嫌犯虽然有钱，但生活简朴不应该买豪车？牵强了些。

    纸质版 23.70元

  • 白夜行

    [日] 东野圭吾 / 刘姿君 / 南海出版公司 / 2013-1-1 / 39.50元
    2020-07-01 读过

    一直保持到最后阶段的核心悬念、精巧的叙事结构、复杂的人物关系网络，这些也许能让一部小说成为一时的畅销书，但要成为公认的经典之作，那还不够。大家对本书的评价如此之高，我想很重要的原因是这个故事的象征意义。“ 我的天空没有太阳，总是黑夜，但并不暗，因为有东西代替了太阳。虽然没有太阳那么明亮，但对我来说已经足够。凭借这份光，我便能把黑夜当成白天。我从来就没有太阳，所以不怕失去。 ”似乎是这种超越了爱情和时间的长久羁绊，引起了人们持久不断的热情讨论。说到底，这世上每个人都是孤独的，都渴望着有一个无论发生了什么都不离不弃，始终陪伴左右，照耀自己的太阳。

    纸质版 30.80元

  • 我决定活得有趣

    2020-06-15 读过

    “我是最喜欢跟年轻人聊天的，因为我想我可以跟他们沟通，我自己心态还算年轻。就是发现很多年轻人，还是跟我有一点代沟，就是我比他们年轻一点。” 心态好，多半还是因为有积累有底气。年轻人没法一开始就学他做性情中人，学了别人也不认账。年轻人还是要有年轻人的样子，当然不是现在那些上街闹事的香港年轻人的样子，而是吃的了苦，充满希望，眼里有光。

  • 算法神探 : 一部谷歌首席工程师写的CS小说

    [美] 杰瑞米·库比卡 / 啊哈磊、李嘉浩 / 电子工业出版社 / 2017-2 / 65
    2020-06-14 读过

    这本书的目标读者当然是初学算法的编程小白，希望借助一些讲解生动的书来入门。但这本书明显犯了喧宾夺主的毛病，算法知识点太少，而且被层层包裹在一个并不生动有趣，反而有些故弄玄虚的儿童故事里。正经学东西的话，知识密度太低；你如果想看故事，那明显是找错了地方。 相比起来，另一本算法入门书——《算法图解》，阅读感受要好得多。

    纸质版 56.60元

  • 小鸟睡在我身旁

    黎戈 / 北京十月文艺出版社 / 2018-1 / 45.00
    2020-06-03 读过

    这本书写了两对母女的生活小事：“我”和女儿，以及母亲和“我”。前者主要是写女儿成长趣事，后者主要是写母亲对家庭的爱护和奉献，间或也提及了一些“我”的生活。 如今当一个专职写字的不知名作家，如果没有家人的支持和其他生活来源，生活大概是清苦的，好在生活的幸福感更多依赖情趣和美感，而不是金钱。这些对生活的细致观察、甘于普通的生活态度，如果能潜移默化的被孩子吸收，也许比一笔庞大的遗产，更能让孩子抵御住往后生活的风雨。

    纸质版 31.40元

  • 氯的滋味

    [法] 巴斯蒂安·维韦斯 / 张鑫佳 / 北京联合出版公司 / 2016-4 / 88.00元
    2020-05-17 读过

    这是一个恋爱故事吗？算不上，刚刚互有好感，暧昧尚未滋生，故事就结束了。情节动人吗？故事平平常常，称之为情节不如说是一段情绪。 作者是法国人，画于2007年，不算太久远。这样的漫画能在西方世界获得好评，我也是有点意外，法国人不是很开放很主动的嘛。几次相遇，互有好感，连名字和电话都没要，这种纯纯的爱恋发生在东方世界才正常。 画风不太精细，好像是一个未经绘画训练的普通人画的日常故事，这样反而让人一开始就不去苛求画工，专注于体会其中的氛围和情绪。 有几个镜头是从第一视角展现的，让人有种投入的感觉，比如：仰泳的时候，整个视野都是玻璃屋顶，然后边上有人从视野边界。啊，是那个我刚才还在满场搜寻却没找到的女孩子，她竟然主动跟我打招呼……

    纸质版 68.60元

  • 伤心咖啡馆之歌

    [美]卡森·麦卡勒斯 / 文泽尔 / 华东师范大学出版社 / 2018-3-30 / 38.00元
    2020-05-14 读过

    这是一个令人悲伤的三角恋故事，我爱你，你爱他，他爱我，首尾相接却永不得满足。 因爱生恨的事情并不少见，天堂和地狱只有一线之隔。艾米丽娅的不知人情世故似乎是后来所有悲剧的开端，她接受了马文·梅西的求婚却实际拒绝了与他共同生活，还狠狠羞辱了他。后来马文·梅西和驼背人离开小镇前干尽了坏事，他们拿走了所有值钱的东西，砸了酿酒的机器，烧了作坊，还做出了一盘有毒的玉米粥，摆在了咖啡馆的柜台上。这些事情有多出格，大概显示了马文·梅西恨的有多深，而此前爱的又有多深。

    纸质版 29.40元

  • 万物：创世

    [德] 延斯·哈德 / 王遥路、陈亮 / 北京联合出版公司 / 2015-4 / 188.00元
    2020-05-08 读过

    这个绘本在豆瓣上评分高达8.9分，得过国际漫画节大奖，不过不是我的菜。 内容很有野心，第一册从宇宙大爆炸开始，一直画到恐龙灭绝，人类登场。画风不是很精细，内容斑杂，暗藏了很多古人探索解释世界的理论和原画，但以西方宗教古典为主，我不是考据狂，并没有进一步探究的想法。如果要说知识，似是而非的绘画，当然比不上详细的文字说明。我前不久读的《万物简史》传递的信息量就要大得多，依我看也更加有趣。

    纸质版 109.60元
